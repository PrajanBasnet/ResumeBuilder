# Resume Builder

A modern and responsive **Resume Builder web application** built using **React.js** and **Tailwind CSS**.  
This project allows users to easily create, edit, and preview a professional resume through a simple and user-friendly interface.

---

## Features

- Interactive resume form
- Live resume preview
- Add and manage:
  - Personal information
  - Education details
  - Work experience
  - Skills and responsibilities
- Clean and responsive UI using Tailwind CSS
- Component-based architecture using React
- Beginner-friendly project structure

---

## Technologies Used

- **React.js** – Frontend library
- **Tailwind CSS** – Utility-first CSS framework
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
